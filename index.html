Install MapFlow as App
Run MapFlow like a desktop app — no browser chrome, just the tool.
🔵 Google Chrome
  1. Click ⋮ (three dots) in the top-right
  2. Go to More tools → Create shortcut…
  3. Check ✅ "Open as window"
  4. Click Create — done!
🔷 Microsoft Edge
  1. Click ··· (three dots) in the top-right
  2. Go to Apps → Install this site as an app
  3. Click Install
💡 Pro tip

Once installed, MapFlow appears in your Start Menu / Applications and opens in its own window — just like Spotify or YouTube Music.
