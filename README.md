# 🧠 Temu → Amazon Mapping Dashboard

A modern web-based tool to map Temu product listings to Amazon ASINs efficiently.

## 🚀 Features

- Upload Excel / CSV files
- Paste Temu product links
- Track mapping progress
- Add ASIN, pricing, brand, seller details
- Export mapped data to Excel

## 🌐 Live Demo

👉 [Temu Mapping Simplifier](https://stevegates24.github.io/Temu-Mapping-Simplifier/)

## 📦 How to Use

1. Upload your Excel file OR paste Temu links
2. Open product pages
3. Map ASIN + details
4. Export final Excel

## ⚡ Tech Stack

- Vanilla JS
- HTML/CSS
- SheetJS (XLSX)

## 📌 Note

This is a client-side tool. No data is stored on servers.
